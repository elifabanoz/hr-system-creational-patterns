# hr-system-creational-patterns
Factory Method and Builder Pattern implementation for HR system. Creates software development teams with different worker types (Blue Collar, White Collar, Managers).

Case:
In this assignment, you are required to analyze the requirements of the HR system which is described
below and implement two of the creational patterns in the design of the system (the appropriate pattern
should be chosen by the students and the reasons should be explained).
Details of the system:
The HR system will be developed for a software company, which has different types of workers. The
information about the workers in the company is given as;
• The company has three types of workers: blue collars, white collars and managers.
• The white-collar workers software specialist or HR specialist.
• Software specialist includes three types of workers: frontend developer, software architect or
backend developers.
• Managers can be technical managers or administrative managers.
The systems will take client’s needs and form software development teams according to requests.
Ex: The client will request to form a software development team including one software architect, one
technical manager, two frontend developers, and seven backend developers.
